# 星雲同期

星雲同期（NebulaSync）は、ローカルフォルダ・NAS・クラウドミラー間でプロジェクトファイルを同期する、架空のクロスプラットフォーム CLI ツールです。競合が発生した場合も、決定的なルールで処理できます。

## 主な機能
- コンテンツハッシュを使った高速な双方向同期
- 再実行可能な解決ログ付きの競合検出
- `.gitignore` 互換の除外ルール
- 変更適用前に安全に確認できるドライランモード
- 差し替え可能なストレージアダプタ（Local、S3 互換、WebDAV）

## よく使うコマンド
- `nebulasync init` : 既定の設定と状態ディレクトリを生成
- `nebulasync sync` : 現在のプロファイルに基づいて同期を実行
- `nebulasync status` : 未同期の変更と差分サマリを表示
- `nebulasync doctor` : 設定・認証情報・アダプタ状態を検証
